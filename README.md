Calculadora de Partidas Rankeadas

Projeto desenvolvido para o desafio Calculadora de Partidas Rankeadas da DIO (Digital Innovation One).

Objetivo
Criar uma função que receba a quantidade de vitórias e derrotas de um jogador, calcule o saldo de vitórias e determine o nível do herói.

Saldo:
vitórias - derrotas

Classificação
Menor que 10 → Ferro
11 a 20 → Bronze
21 a 50 → Prata
51 a 80 → Ouro
81 a 90 → Diamante
91 a 100 → Lendário
101 ou mais → Imortal

Saída
O Herói tem de saldo de {saldoVitorias} está no nível de {nivel}
